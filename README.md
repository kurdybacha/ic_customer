#### Requirements

C++ compiler with C++11 support and make.
Tested on Linux with GCC 5.2.1

#### Compilation

----
* `$  git clone https://github.com/kurdybacha/ic_customer.git`
* `$  cd ic_customer`
* `$  make`

#### Tests
----
* `$  make tests`

#### Main program (main.cpp):
----
* `$  make run`
